How to use the website

Welcome to our website! Most features are only available to registered users. If you are new here head to Sign up in order to register, otherwise you can login here. If you want to find out more about us you can do so by heading to the About page. Functionalities for users are still to be added, but you will also be able to search for plants by filtering specific fields, so keep an eye out for this in the future!

If you encounter a problem on our website, we would appreciate it if you would fill out the form found in the Contact tab. The site is still in a very incipient state and as such there might be a lot of bugs/unimplemented features for now.
